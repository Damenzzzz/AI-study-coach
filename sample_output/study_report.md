## Study-Note Summary

Supervised learning is a machine learning approach where a model learns from input-output pairs.
Each training example contains features, which are the inputs, and a label, which is the expected
output. The model studies patterns in the data so that it can make predictions for new examples.

The dataset is usually divided into training and validation data. The training split is used to fit
the model, while the validation split helps estimate how well the model generalizes. This separation
reduces the risk of evaluating the model on the same data it already memorized.
